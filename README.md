# SaaTz

Веб-приложение для управления учётными записями.

## Описание

Реализована форма со списком записей:

- **Метки** — несколько значений через точку с запятой
- **Тип записи** — Локальная или LDAP
- **Логин** и **пароль**

Валидация обязательных полей выполняется при потере фокуса у текстовых полей и при изменении значения селекта. Данные хранятся в Pinia с сохранением в localStorage. Доступен экспорт списка в JSON.

## Стек

- Vue 3
- TypeScript
- Vite
- Pinia
- PrimeVue
- Tailwind CSS
- Zod

Компоненты организованы по принципам atomic design. Схемы валидации и типы вынесены в отдельные модули.

## Запуск

```bash
npm install
npm run dev
```

Сборка:

```bash
npm run build
```
